# ADR 001 — 엔티티 생성 패턴: 빌더 채택

**날짜**: 2026-06-18
**상태**: 확정

---

## 배경

엔티티 생성 경로를 어떤 패턴으로 통일할지 결정이 필요했다. 팩토리 메서드, 생성자+빌더 혼합 등을 검토했다.

---

## 결정: 생성자 `private` + `@Builder`로 생성 경로 통일

---

## 이유

팩토리/혼합 방식 대비 보일러플레이트 없이 가독성을 확보할 수 있다.

---

## 트레이드오프

빌더가 개방되어 있어 잘못된 조합의 객체 생성이 가능하다. 코드 리뷰로 방어하고, 필요 시 생성자 내부에 런타임 검증을 추가해 보완한다.
